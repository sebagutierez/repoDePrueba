CSS Grid

  • Es un sistema de maquetación web que divide la página en una grilla.
  • Sirve para posicionar los elementos fácilmente.
  • Es el estándar.
  • CASI cualquier navegador lo soporta.
  • FEXBOX Y GRID revolucionaron la programación web al codificar con menos elementos.

Elementos

  • Contenedor
  • Líneas
  • Columnas
  • Celdas = Columnas y Filas
  • Ítems

Activación

display: inline-grid: Establece una cuadrícula con ítems en LÍNEA (ej: inline-block)

display: grid: Establece una cuadrícula con ítems en BLOQUE (ej: block)

grid-template-columns:[col1] [col2] Establece el SIZE de cada columna

grid-template-row:[fila1] [fila2] Establece el SIZE de cada fila

gap:5px Establece el espaciado entre celdas

grid-template-areas: "" Asocia cada elemento a la grilla.

grid-area: "" Asocia el elemento a la celda.

.class:nth-child(nro.item){propiedad:valor} Selecciona un elemento hijo para darle una propiedad y valor.
En HTML: (div class="class")nro.item(/div)
