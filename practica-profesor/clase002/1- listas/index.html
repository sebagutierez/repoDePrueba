Ejemplo de Lista de Descripción

TCP
El Protocolo de control de transmisión es un método (protocolo) utilizado junto con el Protocolo de Internet (IP) para enviar datos en forma de unidades de mensaje, llamadas paquetes, entre ordenadores a través de Internet.
IP
El Protocolo de Internet es el método o protocolo mediante el cual se envían datos de un ordenador a otro en Internet. Cada ordenador en Internet está identificado de forma única por una dirección IP.
FTP
El protocolo de transferencia de archivos es un protocolo utilizado para intercambiar archivos entre ordenadores en Internet.
HTTP
El protocolo de transferencia de hipertexto es el protocolo utilizado para intercambiar texto, imágenes gráficas, sonido, vídeo y otros archivos multimedia en la Web.
